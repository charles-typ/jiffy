# Running Hadoop MapReduce on Jiffy

This guide describes how to run MapReduce Jobs with Jiffy as the underlying 
storage.

## Prerequisites

* Java 1.8 or later
* A running Jiffy cluster
* A cluster with Hadoop 2.9.1 installed (see 
[here](https://hadoop.apache.org/docs/stable/hadoop-project-dist/hadoop-common/ClusterSetup.html) for a guide)
